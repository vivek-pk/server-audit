## Single Binary Build & Daemon Installation

### Build
The entire application compiles to a single static binary:
```bash
go build -o security-scanner ./cmd/scanner
# Or via Makefile:
make build
```

### Installation (`make install`)
The Makefile handles:
1. Copy binary to `/usr/local/bin/security-scanner`
2. Create config directory `/etc/security-scanner/`
3. Install example config to `/etc/security-scanner/config.yml`
4. Install the systemd service file
5. Reload systemd daemon and enable/start the service

### Makefile Targets
```makefile
.PHONY: build test lint clean install uninstall

build:
	go build -ldflags="-s -w" -o bin/security-scanner ./cmd/scanner

test:
	go test -race -timeout 30s ./...

install: build
	install -Dm755 bin/security-scanner /usr/local/bin/security-scanner
	install -Dm644 config.yml /etc/security-scanner/config.yml
	install -Dm644 deploy/security-scanner.service /etc/systemd/system/security-scanner.service
	systemctl daemon-reload
	systemctl enable security-scanner
	systemctl start security-scanner

uninstall:
	systemctl stop security-scanner || true
	systemctl disable security-scanner || true
	rm -f /usr/local/bin/security-scanner
	rm -f /etc/systemd/system/security-scanner.service
	systemctl daemon-reload
```

### systemd Service File (`deploy/security-scanner.service`)
```ini
[Unit]
Description=Linux Security Scanner Daemon
After=network.target

[Service]
Type=simple
ExecStart=/usr/local/bin/security-scanner --config /etc/security-scanner/config.yml
Restart=on-failure
RestartSec=5
StandardOutput=journal
StandardError=journal

[Install]
WantedBy=multi-user.target
```
